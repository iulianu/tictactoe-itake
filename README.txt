This code is written in Scala 2.10.

Download JDK 1.7
Download Scala 2.10 from http://www.scala-lang.org/download/2.10.4.html
Download SBT 0.13 from http://www.scala-sbt.org/release/docs/Getting-Started/Setup.html

Run 'sbt' at the root of the source tree.
At the sbt command prompt type 'test' to check that tests pass.
Let it download the whole internet on your machine ;-)

Type 'run' to run the main class.
